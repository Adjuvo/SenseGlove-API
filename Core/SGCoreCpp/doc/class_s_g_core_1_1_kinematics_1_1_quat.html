Add two quaternions together Using a Hamilton product. NOTE: Q2 is applies first, followed by Q1, as per muliplication order of quaternions!

A rotation by Q1 followed by one of Q2 requires to operation Q2*Q1, using the Hamilton Multiplication. https://en.wikipedia.org/wiki/Quaternion
