static Quat SGCore.Kinematics.Quat.LookRotation_XYZ ( Vect3D  X,
Vect3D  Y,
Vect3D  Z 
)
static

Calculate the Quaternion that, when applied to (1, 0, 0), (0, 1, 0) and (0, 0, 1) will result in teh vectors X, Y, Z.

Parameters
XIn Orignal function - Side, calculated from Cross Up(input)-FWD, Unity (2D) Left/Right
YIn Orignal function - UP, re-calculated from Cross SIDE-FWD, Unity (2D) Up/Down
ZIn Orignal function - Forward, Unity Z (towards screen)

Based on the implementation at https://gist.github.com/aeroson/043001ca12fe29ee911e or https://answers.unity.com/questions/467614/what-is-the-source-code-of-quaternionlookrotation.html, refactored somewhat to make it more readable, and added comments that reflect my own hand solved variables based on the sources from Quat.Rotate.
