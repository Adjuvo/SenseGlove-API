Detailed Description

Interface for the Android Library

Since there is no Native Bluetooth SDK for Android, we have to use .aar files for now. There is no way to import such methods in C#, nor easily in C++. So instead, our Unity back-end handles Sending/Receiving to the API, and data ends up in this class.
