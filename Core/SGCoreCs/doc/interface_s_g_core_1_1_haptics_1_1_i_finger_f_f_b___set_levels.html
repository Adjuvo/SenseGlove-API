Set the Force-Feedback value of a particular finger to a specific level

Parameters
level01Value will be clamped between [0...1], where 0.0f means no Force-Feedback, and 1.0 means full force-feedback.
fingerThe finger to which to send the command.

Devices that 'only' have on/off FFB will treat any value > 0.0 as 1.0.

Returns
Returns true if the command was sucessfully added to the queue. If false, the device might not support the chosen finger(s) or have the appropriate firmware.

bool SGCore.Haptics.IFingerFFB_SetLevels.QueueFFBLevels ( float[]  levels01)

Queue a list of force-feedback levels, between 0.0 and 1.0. Your list should be sorted from thumb to pinky.

Parameters
levels01Array containing the Force-Feedback levels, from 0.0 (no FFB) to 1.0. A value < 0.0f means this finger won't be updated.

Devices that 'only' have on/off FFB will treat any value > 0.0 as 1.0.

Returns
Returns true if the command was sucessfully added to the queue. If false, the device might not support the chosen finger(s) or have the appropriate firmware.
