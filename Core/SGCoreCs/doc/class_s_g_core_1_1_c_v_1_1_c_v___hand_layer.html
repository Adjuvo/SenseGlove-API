Detailed Description

An abstract layer with which to interface for CV models, specifically for hand and finger tracking.

I'm going to assume for now that we only use one device subtype at the same time;e.g. 2x Nova 1.1, or 2x Nova 2.0. But never 1.1 & 2.0 at the smame time.
