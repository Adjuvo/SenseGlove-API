The type of connection this device is using

Enumerator
Unknown 

This device has been disconnected, or this version of SGCore does not recognize the type of connection.

Serial 

USB Serial port connection. Used for DK1 SenseGloves

BluetoothSerial 

Blueooth connection through Serial ports. Used for Wireless Kits and early Nova prototypes.

Bluetooth Connection Via Android/Java SDK
